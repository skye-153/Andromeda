Andromeda
Manager/Planner app which runs locally on your system
Made using NextJS,React,Lowdb for lightweight local backend,NodeJS and Nativefier(wrapper for electron).
Needs NodeJS to work. https://nodejs.org/en/download
To start:
Double click on the start-app.bat to start the app.
To move to desktop
1.	Right click bat file.
2.	"Show more options".
3.	Create Shortcut.
4.	Give it any name and icon that you want(or the icon.ico in the files).
5.	Move or drop it onto the desktop.
